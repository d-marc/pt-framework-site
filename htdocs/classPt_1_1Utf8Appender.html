Detailed Description

The Utf8Appender is a single-pass output iterator, to append unicode characters to an UTF-8 encoded string. It can be used with standard C++ algorithms.

std::string s;
Pt::Utf8Appender a(s);
Char oe = 0366;
std::fill_n(a, 3, oe);
assert(s, "\303\266\303\266\303\266");
