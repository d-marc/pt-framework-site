Supported Platforms

Microsoft Windows
Windows XP, Vista, 7, 8, RT and CE are supported using the Visual Studio (2005 - 2013) compilers. Support for MinGW 3.x/4.x is experimental.
Apple
Mac OS-X version 10.6 and higher is supported using the compiler provided by the Xcode packages. Experimental support for iOS is also available.
Linux
Linux is supported on x86, x86-64, PowerPC, arm, mips, sh, avr-32 and sparc CPUs using the gcc based compilers 3.x and higher. The LSB platform version 3.x and newer is supported.
Classic Unix
Solaris v10 and newer is supported using the Sun Studio compiler and IBM AIX v5 and newer are supported using the XLC compiler suite. The gcc compiler is supported on both platforms.
QNX Neutrino
The QNX NTO real-time operating system version 6.3.x and newer is supported using the included QCC compiler.
